【 説明 】  
  このソフトは、Webサーバーを用意しなくてもWiki書式の文書を
  扱うことを目的としています。
  また、スケジュールの記載もWiki書式で行うことができ、メモと
  スケジュールを関連づけることも可能です。

  このソフトの開発を始めたことで、PersonalWiki2の開発は終了
  します。

  ご意見等あれば、Project D.E.VのWikiかメールでお願いします。
  Wikiは以下のURLからいくことができます。
  http://aquaspac.hp.infoseek.co.jp/
  メールも上記URLを開き、よく読んでからメールをお願いします。
  メールの場合、必ず表題に[PersonalWiki]を入れてください。
  入れていただかないと、スパムメールフィルタで即ゴミ箱に
  行く可能性があります。

【Windows Vista,Windows 7での動作について】
  作者の手元にはWindows XP SP3の環境しかありません。
  その為、Vista,7での動作は確認できませんのでご了承下さい。

【 最後に 】  
  最後に、ローカルWikiを作成したきっかけは、Wikiを自分で作っ
  てみたかったことと、ToDoやメモを気軽に取りたかったことがあ
  ります。
  このソフトでようやく自分が求めていたものが形になりそうです。
